GA4 events

OSP sends various events in the Google Analytics 4 (GA4) format that you can use to measure different user interactions or occurrences in your mobile and web apps.

You can use these events to track when a user loads a page, clicks a link, or completes a purchase, or to measure system behavior, such as when the app crashes.

What is GA4

GA4 is the latest version of Google Analytics that will replace the existing Universal Analytics (UA) version.

GA4 lets you:

  • Collect data from both mobile and web apps to better understand the user journey.

  • Control what personal data is collected with improved user privacy settings.

  • Generate engaging reports with enhanced visualization and reporting features.

  • Integrate with media platforms directly to help drive actions in your mobile and web apps.

How user interactions are captured in GA4

A major difference between UA and GA4 is how user interactions are captured. In UA, user interactions are captured as hits of different types like pageview, event, social.

GA4 however, uses a different measurement model and captures every user interaction as an event. This means that all hit types in UA translate to events in GA4. For example, a user interaction that is captured as a pageview in UA is captured as an event in GA.

The table below shows the different hit types in UA that are captured as events in GA4:

Hit types in UA Corresponding measurement in GA4
pageview event
event event
social event
transaction/eCommerce event
user timing event
exception event
app/screen view event

By using the same measurement model as Google Analytics Firebase where all user interactions are captured as events, GA4 lets you track data from mobile and web apps in the same property without the need to set up different properties for different platforms.

GA4 event structure

In UA, an event is a hit type and has an associated category, action, and label. In GA4 however, events have no category, action, or label. Every hit is an event that can contain certain parameters that provide additional information about the user interaction (event).

Because the data model and event structure in GA4 are significantly different from UA, you must implement new event logic to use the new GA4 events.

Switching to GA4

All UA events in OSP will be deprecated and replaced with new GA4 events. For more information and the latest updates on GA4, see the Google Analytics Help website.
