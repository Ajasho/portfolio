Product promotion click

Event name select_item_promotion
Event type Custom
Event trigger A customer clicks the promotion text on a product card or a product details page.

Event parameters

This event is based on the Google Analytics 4 (GA4) specifications and includes the following parameters:

Parameter Value
event select_item_promotion
id The retailerProductId of the product.
name The name of the product.
index The position (0,1,2,3,4…) of the product on the page on which the product is dsiplayed to the customer.
list The page type and name of the section where the product is displayed, in page type:section name format.

For example, for a product in the Favorites section on the Promotions page, the list parameter value is displayed as promotions: favorites.
promotion_id The retailerPromotionId of the promotion.
promotion_name The name of the promotion.

Example

{
   "event":"select_item_promotion",
   "id":"296967011",
   "name":"Soreen 5 Individual Malt Lunchbox Loaves 150g",
   "index":5,
   "promotion_id":"a4ff4595-fd5c-4c86-bd63-6c96b0d79487",
   "promotion_name":"Now £1.25, was £1.75",
   "list":"logged-in-homepage: other"
}
