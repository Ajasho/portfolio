Work with recipes

Use the recipe-related content types in Contentful to create the following elements of the recipe offering in your online store:

  • Recipe details
  • Recipe collection
  • Recipe home page
  • Store home page card

Recipe details

A recipe details page is a space to display detailed information about a specific recipe, for example, steps required to prepare the dish, list of ingredients, cooking time.

This page displays when a customer selects a specific recipe in your online store.

See Recipe details for more information on how to create the recipe details entry.

Recipe collection

A recipe collection is a space where you can collect multiple recipes that belong to one category or are linked to a particular occasion. For example, you can create a recipe collection called Brilliant bakes, Vegan recipes, or Perfect pasta.

A recipe can be associated with multiple collections.

See Recipe collection for more information on how to create the collection entry.

Recipe home page

The recipe home page is a main space for displaying recipes, recipe collections, and recipe collection groups.

Customers can access the recipe home page:

  • From the Recipes tab in the online store
  • From the Browse pages in OSP mobile apps

You can create and configure the recipe home page in Contentful using the following elements:

  • Static banner image - The main image for your recipe home page.

  • Configurable entities - A set of entities that you can configure and display on the recipe home page. You can configure your recipe home page using the following entities:

    • Recipe group - a group of recipes that you want to present together.
    • Collection group - a group of recipe collections.
    • Expanded collection - a space to highlight a specific recipe collection and promote some recipes that belong to that collection.

See Recipe home page for more information on how to create and configure the home page for your recipe offering.

Store home page card

The store home page card is a card to display and navigate to recipe content from your store’s home page. These cards appear in a separate section called Recipes on your store’s home page. When customers click a card, they are taken to a specific page within your recipe offering. You can display a maximum of 6 cards on the store’s home page.

You can link the store home page card to either of the following pages:

  • Recipe home page
  • Recipe details
  • Recipe collection

See Store home page card for more information on how to create and configure store home page cards for your recipe offering.
