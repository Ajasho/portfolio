Recipe details

This page displays detailed information about the specific recipe.

This page also displays the Shop for ingredients button that customers can use if they decide to buy the ingredients of this recipe.

  • Create a new recipe details page
  • Configuration

Create a new recipe details page

To create a new recipe details page:

  1. On the Contentful landing page, go to the Content tab.

  2. Click Add Entry and select Recipes : Recipe details.

  3. Complete all required fields - see the Configuration section for guidance.

  4. Click Publish.

Configuration

When creating a new recipe details page, there are several sections (see note below) to complete.

The sections are not reflected in the interface and are only used to better describe the purpose and context of each field.

  • Recipe identification
  • Recipe image
  • Recipe details
  • SEO
  • Retailer content ID

Recipe identification

This section includes the following required fields:

  • Title - specify a title for your recipe. The title is displayed as the name of the recipe in your online store.

  • Description - Add a description for the recipe. The description is displayed as the description of the recipe in your online store.

If your online store supports multiple languages, you must specify the Title and Description in each of those languages.

  • Slug - specify a slug for your recipe. The slug is a lowercase string with hyphens in place of spaces, and displays in the browser URL.

    If empty, the slug is sourced from the Title field.

Recipe image

This section includes the following required fields:

  • Image - add an image for the recipe.

    • Use an image with a minimum size of 1280px × 1280px. See image guidelines for the detailed specification.
    • If your online store supports multiple languages, you only need to upload the image to the default locale. The image will automatically display across all available locales.

  • Image alt text - specify an alt text for the recipe image.

    If your online store supports multiple languages, you must specify the Image alt text in each of those languages.

Recipe details

This section includes the following fields:

  • Published date - specify the date the recipe was first published in your service. This is a required field.

  • Number of servings - specify the number of servings in the recipe.

  • Calories per serving - specify the number of calories in one serving of the recipe.

  • Preparation time (minutes) - specify the time required to prepare the ingredients.

  • Cooking time (minutes) - specify the time required to cook the recipe.

  • Total time (minutes) - specify the total time required to prepare and cook the recipe. This is a required field.

  • Author - specify the author of the recipe. This may be the name of the person or organization that authored the recipe. This is a required field.

    You can either create a new author for your recipe or link an existing author that you have created before.

    To create a new author, see Author.

    To link an existing author to your recipe:

    1. In Author field, click + Add content and then select Add existing content.
    2. Select the required author from the list.

  • Ingredients - specify the list of ingredients required to complete the recipe. This is a required field.

    Before you can specify the list of ingredients, make sure you have the entries for all units of measurement that your recipe requires available in Contentful. See Create a Unit of measurement entry for more information.

    To add an ingredient:

    1. In the Ingredients field, click + Add ingredient.

    2. Specify the Quantity of the ingredient required for the recipe. For example, 1.5, 10, or 500.

      This is an optional field and is tagged as @quantity.

    3. Select the Unit of measurement that the ingredient’s quantity is expressed in.

      This field is an optional field and is tagged as @uom.

    4. Add the ingredient, either as a free-form text, for example, ‘a pinch of salt’, or by using the @quantity and @uom tags before the ingredient. For example, to show ‘2kg flour’, type ‘@Quantity@uom flour’.

      The Ingredient preview displays how the ingredient will show in the recipe.

    5. To continue with a system-generated search term, click Save Ingredient. OSP will use a system-generated search term to present results to customers.

      Alternatively, you can use a custom Search term for this ingredient:

      1. Clear the Use generated search term checkbox.

      2. Specify the search term for this ingredient. OSP uses this search term to present results to customers. This is a required field.

        You can enter the specific product ID in this field.

      3. Click Save Ingredient.

      You can see the search term, whether custom or system-generated, next to each ingredient in the list of ingredients.

  • Method - specify the steps required to complete this recipe.

    To add a simple method:

    1. In the Method field, click + Add method and select Add simple method.
    2. Enter the description of the step. This is a required field.
    3. Click + Add step to add a new step.
    4. Repeat steps 2 and 3 until you have entered all steps in the recipe.
    5. Click Save Changes.


    For more complex recipes, you can group steps for the different preparation processes. For example, if baking a cake, you could have separate steps for baking the sponge and making the icing.

    To add a grouped method:

    1. In the Method field, click + Add method and select Add grouped method.
    2. Specify a Group heading for the recipe section (for example, ‘Prepare the ingredients’). This is a required field.
    3. Enter the description of the step. This is a required field.
    4. Click + Add step to add a new step to the recipe.
    5. Repeat steps 3 and 4 until you have entered all steps that belong to this grouped method.
    6. Click Save Changes.
    7. Repeat steps 1 through 6 to add another grouped method in the recipe.

  • Regions - specify the regions you want this recipe to be available in. If empty, the recipe is available across all regions you operate your online store in.

  • Search terms - specify the list of terms your customers may use to search for the recipe. OSP uses these terms to tailor search results when the customer searches a recipe from the search bar on the recipes home page.

  • Structured data category - the meal type or the course that your recipe reflects, for example, ‘dinner’, ‘dessert’.

    You can either create a new or link an existing structured data category for your recipe.

    To create a new structured data category, see Structured data category.

    To link an existing structured data category to your recipe:

    1. In Structured data category field, click + Add content and then select Add existing content.
    2. Select the required structured data category from the list.

  • Structured data keywords - specify other terms that you want to link to the recipe, for example, ‘summer’, ‘halloween’, or ‘quick’, ‘easy’. Search engines like Google use these terms to tailor search results when the customer searches for a recipe on the internet.

SEO

Use this section to make your recipe findable by the search engines.

This section includes the following fields:

  • SEO: No index? - select whether you would like to index the recipe. Indexed recipes appear in the Google search results. Select ‘False’ to index the recipe.

  • SEO: Canonical URL - specify the canonical URL for the recipe. If multiple URLs are used to access this recipe, the canonical URL is the one crawled by Google and other search engines.

  • SEO: Meta title - specify the page title that displays in the browser’s title bar or in the page’s tab. If empty, the ‘Recipe title tag rule’ is used.

    See ‘Recipe title tag rule’ in SEO settings for more information.

  • SEO: Meta description - specify a short description for the recipe page that displays in search results. If empty, the ‘Recipe description tag rule’ is used.

    See ‘Recipe description tag rule’ in SEO settings for more information.

  • SEO: Meta keywords - specify the list of keywords to give legacy search engines more information about the content of this recipe details page.

Retailer content ID

Use this section if you edit and add recipe content through the OSP APIs:

  • Retailer content ID - enter an ID to identify this entry when using the Recipes API.
